About

iConnect Plus is an Extension for Bluedoor's iConnect intranet.

Features

  • Ability to edit posts as HTML
  • Syntax highlighting for Code Snippets within the portal
  • Themes: Timber, Dark Leather
  • Ability to specify custom CSS
  • Ability to specify preferred sort method for posts (e.g. Recent / Alphabetical)
  • Chrome Notifications icon
